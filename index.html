The map below allows users to easily locate the nearest hopistal from libraries in the Tacoma area. When a hosptial symbol is clicked on a pop up will appear with the hosiptal name and address. When library symbol is clicked a popup will appear with the library name, the name of closest hospital to the library, address of hopsital, and distance to the hospital. The interface allows the users to easily locate the nearest hospital, understand the distance, and have the address to the hospital.


Sources: Pierce County Public Health Care Facilities and Libraries
